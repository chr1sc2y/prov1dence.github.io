Support Vector Machine

Cost Function

$ \min{\theta} C\sum{i=1}^m(y^{i}cost1(\theta^T X^i) + (1 - y^{i})cost_0(\theta^T X^i)) + \frac{1}{2}\sum{i = 0}^m \theta_j^2 $

  • C: not too large

Decision Boundary

Margin

$\min{\theta} \frac{1}{2} \sum{j=1}^n \theta_j^2 = \frac{1}{2}(\theta_1^2 + \theta_2^2) = \frac{1}{2}\sqrt{\theta_1^2 + \theta_2^2} = \frac{1}{2}||\theta||^2 $

Recommender System

Anomaly Detection

  1. choose features $x_i​$ which might be indicative of anomalous examples

  2. fit parameters $\mu_1, \mu_2, ... , \mu_n, \sigma_1^2, ... , \sigma_n^2 ​$

  3. $ \muj = \frac{1}{m} \sum{i=1}^{m} x_j^{i} ​$

  4. $ \sigma^2 = \frac{1}{m} \sum_{i=1}^{m}(x_j^{(i)} - \mu_j)^2 ​$

  5. compute $p(x)​$

  6. $ p(x) = \prod{j=1}^{n} p(x_j;\mu_j,\sigma_j^2) = \prod{j=1}^n(x \sim N(x;\mu, \sigma^2)) $

Gaussian (Normal) Distribution

$ x \sim N(x;\mu, \sigma^2) = \frac{1}{\sqrt{2\pi}\sigma} exp(-\frac{(x-\mu)^2}{2\sigma^2}) ​$
